Terminal cheat sheet
--------------------

Ctrl + Alt + F1-6   Open one of the 6 terminals
Ctrl + Alt + F7     Back to the original GUI terminal

Ctrl + A    go to start of the line
Ctrl + B    move the cursor backward 1 character
Ctrl + C    stop current running command
Ctrl + D    logout
Ctrl + E    go to end of the line
Ctrl + F    move the cursor forward 1 character
Ctrl + K    cut right of the cursor
Ctrl + L    clear terminal
Ctrl + Q    show hidden screen output (XON)
Ctrl + R    reversed history search
Ctrl + S    stop screen output (XOFF)
Ctrl + T    swap two characters left/right to the cursor
Ctrl + U    cut left of the cursor
Ctrl + W    cut word on the left of the cursor
Ctrl + Y    paste

!!          repeat last command
!!:p        see last command
!cmd        repeat last 'cmd' command
!cmd:p      see last 'cmd' command

~           current user $HOME directory
